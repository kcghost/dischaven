dischaven
=========
Utility for ripping DVDs and Blu-Rays to high quality mp4 format.
Tries to "do what you want" as much as possible.

Install
-------
Just run:
```
make install
```

Uninstall with:
```
make uninstall
```

Usage
-----
Run `rip -h` to see full usage details.
